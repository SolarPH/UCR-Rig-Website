Setting up the Importer


The Importer Wizard is a tool for UCR that enables the user to create any character of any preset type at any location and in any heirarchy on your active scene.

Fields:

1. Character Name
    - It is the character name you assign to a character model before import. The whole rig will be renamed to the character name you chose.

2. Skin Path
    - It is the file path of your skin file. It will be set on your character's mesh upon creation.
    NOTE: The Importer would not run if the skin is an invalid file. The "SKIN" side will turn RED if the skin file cannot be used.

3. SSS / Subsurface Scattering File Path
    - It is the file path for the skin mask used by the Subsurface Scattering Layer. See the image below for the comparison of the Skin and Subsurface Scattering layers look like.
    NOTE: The importer would not run if the SSS File is invalid. The "SSS" side will turn RED if the SSS file cannot be used.


4. V3 Type
    - It is the type of face you want your character to have.
        * Square Mouth
        * Rounded Square Mouth
    NOTE: Some legacy V2/V1 facial rigs may be available on the pack.

5. Character Type
    - Here you can choose between the Steve(4px) arms or the Alex(3px arms) body model.

6. Mesh Type
    - Here you can choose between 3 UVW Types:
        * Freesize - Traditional UV Mesh Setup which can contain 64px and larger skin format.
        * High-Poly - Pinpoint UV Mesh Setup which can only contain 64px skin format, specialized for Extrusion Compatibility.
        * Extra High Poly - Pinpoint UV Mesh Setup which can only contain 128px skin format, specialized for Extrusion Compatibility.

7. Include Bones Toggle
    - You can choose if you want to include Body Control set for the character. You can turn this off in cases of you already had a set of bones to use for the character being created, or if you want to only create a character ready for extrusion.

8. Cape Type
    - Here you can choose what cape you want to be included with your character.
        * Regular Cape - This is the cape model commonly granted during MineCon.
        * Optifine Cape - This is the cape model included when using Optifine.

9. Importer Code
    - The Current Code field shows what is the importer code based on its current settings. Inputting a code on the New Code Field then applying it using a special toggle will instantly change your importer settings. Be aware that it will also change the character name field to whatever name was past the Underscore part of the code.

10. Run
    - This special toggle will attempt to create a character based on your importer settings (depending on the conditions stated on this page in numbers 2 and 3). This will also channge the character's position, rotation, and scale, relative to the importer's coord settings.


11. Select After Run
    - This setting indicates if what will be selected on the scene after a character has been made.
        * None: This setting deselects everything from the scene.
        * This Importer: This will make the selection stay at the current importer.
        * Generated Binder: Selects the generated character root after the import.
